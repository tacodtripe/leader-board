# Webpack Boilerplate

This is a template to be the start point for future projects where webpack is involved.

## Built With

- Javascript
- HTML
- CSS

## Getting Started - How to deploy on your machine

- Clone the project
- run `npm start` at your command line

### Prerequisites

- Text Editor (VSCode for example)
- Browser
